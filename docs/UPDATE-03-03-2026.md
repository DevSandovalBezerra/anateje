# UPDATE 03-03-2026

## Resumo
Nesta atualizacao foi realizada a consolidacao do fluxo financeiro para reduzir redundancia entre menus de contas e lancamentos, mantendo compatibilidade com rotas/permissoes legadas.

Tambem foi ajustada a ordem do sidebar administrativo para priorizar acesso rapido a permissao e usuarios.

## Entregas realizadas

1. Unificacao de fluxo financeiro (contas x lancamentos)
- Definido `financeiro/lancamentos` como fluxo principal de contas a pagar/receber com parcelamento.
- Rota legada `financeiro/contas` mantida como alias para `financeiro/lancamentos`.
- Compatibilidade de permissao entre `financeiro.contas` e `financeiro.lancamentos` para nao quebrar perfis existentes.

Arquivos:
- `includes/page_router.php`
- `includes/rbac.php`

2. Padronizacao de nomenclatura no menu e titulos
- Rotulos atualizados para reduzir ambiguidade:
  - `Financeiro - Contas a Pagar/Receber`
  - `Financeiro - Contas de Caixa/Banco`
  - `Financeiro - Contas de Apoio`
- `financeiro/contas` removido da sequencia visual principal do sidebar para evitar item duplicado.

Arquivos:
- `includes/sidebar.php`
- `index.php`

3. Reordenacao do sidebar administrativo
- Itens `Permissoes` e `Usuarios` movidos para logo abaixo de `Dashboard Admin`.

Arquivo:
- `includes/sidebar.php`

## Validacoes executadas
- `php -l includes/page_router.php`
- `php -l includes/rbac.php`
- `php -l includes/sidebar.php`
- `php -l index.php`

## Resultado esperado
- Navegacao financeira mais coerente, sem duplicidade de entrada para o mesmo fluxo operacional.
- Menor risco de confusao entre "contas a pagar/receber" e "contas de caixa/banco".
- Melhor usabilidade no admin com atalhos de permissao e usuarios no topo da navegacao.
