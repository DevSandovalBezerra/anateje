# UPDATE 02-03-2026

## Resumo
Nesta atualizacao foi concluida a base de implantacao do PRD/KIT MVP (sem integracoes externas), com foco em:
- paginas publicas faltantes
- autenticacao com rate limit e recuperacao de senha
- modulo admin/permissoes funcional
- ajuste de coerencia visual no tema claro e contraste institucional da sidebar

## Entregas realizadas

1. Paginas publicas MVP
- Criadas paginas publicas:
  - `frontend/public/sobre.php`
  - `frontend/public/beneficios.php`
  - `frontend/public/eventos.php`
  - `frontend/public/blog.php`
  - `frontend/public/contato.php`
  - `frontend/public/filiacao.php`
- Criado layout compartilhado:
  - `frontend/public/_layout.php`
- Criado stylesheet dedicado:
  - `assets/css/public-pages.css`
- Home publica atualizada para navegar entre as paginas:
  - `frontend/public/home.php`
- Roteamento do `index.php` ajustado para permitir acesso publico em `public/*`.

2. API publica para captura de lead e contato
- Novo endpoint:
  - `api/v1/public.php`
- Acoes:
  - `action=lead_capture` (pre-cadastro de filiacao)
  - `action=contact` (mensagem de contato)
- Tabelas de apoio criadas automaticamente:
  - `public_leads`
  - `public_contacts`

3. Autenticacao e seguranca
- `api/auth/login.php` reestruturado para:
  - rate limit de login por email/IP
  - solicitacao de recuperacao de senha
  - validacao de token de reset
  - redefinicao de senha por token
- Tabelas de seguranca criadas automaticamente:
  - `auth_login_attempts`
  - `password_reset_tokens`
- Login frontend atualizado para fluxo novo:
  - `frontend/auth/login.html`
  - `assets/js/api-config.js`
- Novas telas:
  - `frontend/auth/forgot-password.html`
  - `frontend/auth/reset-password.html`

4. Admin - permissoes
- Novo endpoint:
  - `api/v1/permissions.php`
- Funcionalidades:
  - listagem de perfis/permissoes
  - CRUD de perfis
  - atribuicao de permissoes por perfil
  - sincronizacao de permissoes JSON no perfil
- Tela admin implementada:
  - `frontend/admin/permissoes.php`

5. Design system e contraste (tema claro)
- Ajuste de tokens do tema `light` para fundo realmente claro:
  - `assets/css/themes.css`
- Atualizacao formal do design system:
  - `docs/system.design.md` (v1.1)
- Sidebar com faixa de marca escura institucional no topo:
  - `includes/sidebar.php`
  - `assets/css/colors.css`
- Padrao escuro aplicado em hover/active da sidebar e badge de perfil:
  - `assets/css/themes.css`
  - `assets/css/colors.css`
  - `includes/layout.php`
